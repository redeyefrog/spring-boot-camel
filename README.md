# Apache Camel

Notice
--
* camel-http4 was renamed in Camel 3.x 

    Camel 2.x
    ```xml
    <dependency>
        <groupId>org.apache.camel</groupId>
        <artifactId>camel-http4-starter</artifactId>
    </dependency>
    ```
    Camel 3.x
    ```xml
    <dependency>
        <groupId>org.apache.camel.springboot</groupId>
        <artifactId>camel-http-starter</artifactId>
    </dependency>
    ```
